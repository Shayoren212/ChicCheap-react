# ChicCheap React Project

A Vite + React prototype for ChicCheap — a smart fashion price-comparison and discovery app.

## Run locally

```bash
npm install
npm run dev
```

## Structure

```text
your-project/
  DESIGN.md
  src/
    main.jsx
    App.jsx
    styles/globals.css
    components/
    pages/
```

The prototype includes: landing/search page, login/register, filters modal, item details, notifications, profile/settings, and about page.
